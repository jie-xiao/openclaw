# 任务派发模板

## 标准任务派发

```
📋 任务名称：[任务标题]
👤 负责人：[名字]
📅 截止时间：[日期]
📝 任务描述：[详细说明]
🔗 相关资源：[链接/文件]
✅ 验收标准：[如何算完成]
```

## 紧急任务派发

```
🚨 紧急任务！
📋 任务名称：[任务标题]
👤 负责人：[名字]
⏰ 要求完成时间：[具体时间]
📝 原因：[为什么紧急]
📝 任务描述：[详细说明]
✅ 验收标准：[如何算完成]
```

## 部门内部任务

```
📋 部门任务：[任务标题]
👤 负责人：[部门成员]
📅 截止时间：[日期]
📝 任务描述：[详细说明]
🔗 部门：[CTO/CFO/COO/CMO/CPO/CHRO]
```

## 跨部门协作任务

```
🔄 跨部门任务：[任务标题]
👤 主负责人：[名字]
👥 协作部门：[部门A、部门B]
📅 截止时间：[日期]
📝 任务描述：[详细说明]
📋 各部门分工：
  - [部门A]：[具体任务]
  - [部门B]：[具体任务]
✅ 验收标准：[如何算完成]
```

# 汇报模板

## 日常汇报

```
📊 [成员名字] 日常汇报

📅 日期：[YYYY-MM-DD]

✅ 今日完成：
1. [任务1]
2. [任务2]

📋 进行中：
1. [任务名称] - 进度 XX%

🔍 遇到问题：
- [问题描述]

📝 明日计划：
1. [任务1]
2. [任务2]
```

## 周报

```
📊 周报 - [成员名字]

📅 日期：[周范围，如 2026-03-09 ~ 2026-03-15]

✅ 本周完成：
| 任务 | 状态 | 备注 |
|------|------|------|
| 任务1 | ✅ | 已完成 |
| 任务2 | ✅ | 已完成 |

📋 进行中：
| 任务 | 进度 | 预计完成 |
|------|------|----------|
| 任务3 | 50% | 周五 |

🔍 问题与需求：
- [问题1]
- [需求1]

📝 下周计划：
1. [任务1]
2. [任务2]
```

## 项目汇报

```
📋 项目汇报 - [项目名称]

📅 日期：[YYYY-MM-DD]
👤 汇报人：[名字]

📊 整体进度：XX%

✅ 已完成：
- [功能点1]
- [功能点2]

🔄 进行中：
- [功能点3] (进度XX%)

📋 待完成：
- [功能点4]
- [功能点5]

🔍 风险与问题：
- 风险1：[描述]
- 问题1：[描述]

💡 需要支持：
- [支持需求]
```
