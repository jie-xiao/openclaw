# 观复阁流程图汇总

## 1. 任务派发流程

```
┌─────────────┐
│    阁主     │ ← 任务来源
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  判断紧急？  │
└──────┬──────┘
       │
   ┌───┴───┐
   │       │
  是      否
   │       │
   ▼       ▼
┌─────────┐ ┌─────────────┐
│ 立即@   │ │ CEO分配    │
│ 相关成员 │ └──────┬──────┘
└─────────┘        │
                   ▼
            ┌─────────────┐
            │ 判断类型    │
            └──────┬──────┘
                    │
        ┌─────────┼─────────┐
        ▼         ▼         ▼
    ┌───────┐ ┌───────┐ ┌───────┐
    │单部门  │ │跨部门  │ │日常  │
    │任务    │ │任务    │ │任务  │
    └───┬───┘ └───┬───┘ └───┬───┘
        │         │         │
        ▼         ▼         ▼
    ┌─────────┐ ┌─────────┐ 直接
    │部门负责 │ │CEO协调 │ 派发
    │人分配  │ │跨部门   │ 部门
    └─────────┘ └─────────┘
```
