# CVSS 漏洞评分参考

## 评分等级

| 等级 | 分数范围 | 颜色 | 描述 |
|------|---------|------|------|
| 严重 (Critical) | 9.0-10.0 | 🔴 | 需要立即修复 |
| 高危 (High) | 7.0-8.9 | 🟠 | 应该优先修复 |
| 中危 (Medium) | 4.0-6.9 | 🟡 | 根据情况修复 |
| 低危 (Low) | 0.1-3.9 | 🟢 | 可选修复 |
| 无 (None) | 0.0 | ⚪ | 不适用 |

## 评分维度

### 1. 基本指标 (Base Score)

**攻击向量 (AV)**:
- 网络 (N): 可远程利用
- 邻近 (A): 需要物理接近或 LAN
- 本地 (L): 需要本地访问
- 物理 (P): 需要物理接触

**攻击复杂度 (AC)**:
- 低 (L): 容易利用
- 高 (H): 需要特定条件

**所需权限 (PR)**:
- 无 (N): 无需权限
- 低 (L): 需要基本用户权限
- 高 (H): 需要管理员权限

**用户交互 (UI)**:
- 无 (N): 无需用户操作
- 需要 (R): 需要用户操作

**影响范围 (S)**:
- 未改变 (U): 只影响当前组件
- 改变 (C): 影响周边组件

### 2. 示例评分

```
远程代码执行 (RCE):
AV:N + AC:L + PR:N + UI:N + S:C → 9.8 (严重)

SQL 注入:
AV:N + AC:L + PR:N + UI:N + S:U → 8.6 (高危)

存储型 XSS:
AV:N + AC:L + PR:N + UI:R + S:C → 6.1 (中危)

信息泄露:
AV:N + AC:H + PR:L + UI:N + S:U → 4.3 (低危)
```

## 常见漏洞评分

| 漏洞类型 | 典型评分 |
|---------|---------|
| 未授权远程代码执行 | 9.8 |
| SQL 注入 (无数据泄露) | 8.6 |
| 存储型 XSS | 6.1 |
| 反射型 XSS | 5.3 |
| CSRF | 4.3 |
| 敏感信息泄露 | 5.0 |
