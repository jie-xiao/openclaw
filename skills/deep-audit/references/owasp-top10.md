# OWASP Top 10 漏洞类型

## A01:2021 - 失效的访问控制

用户可以访问超出其权限范围的资源。

**常见位置**:
- URL 参数修改
- 权限检查缺失
- IDOR (不安全的直接对象引用)

**防御措施**:
- 最小权限原则
- 访问控制检查
- 禁用目录浏览

## A02:2021 - 加密失败

敏感数据未加密或加密不当。

**常见问题**:
- 使用弱加密算法 (MD5, SHA1)
- 硬编码密钥
- 缺少 SSL/TLS

**防御措施**:
- 使用现代加密算法 (AES-256, RSA-2048)
- 安全存储密钥
- 使用 HTTPS

## A03:2021 - 注入

用户输入被解释为代码执行。

**类型**:
- SQL 注入
- 命令注入
- LDAP 注入
- XPath 注入

**防御措施**:
- 参数化查询
- 输入验证
- 最小权限原则

## A04:2021 - 不安全的设计

应用程序架构或设计缺陷。

**常见问题**:
- 业务逻辑漏洞
- 缺少安全设计模式

**防御措施**:
- 威胁建模
- 安全设计模式
- 代码审查

## A05:2021 - 安全配置错误

系统或应用程序配置不当。

**常见问题**:
- 默认凭据
- 错误消息泄露信息
- 调试模式开启

**防御措施**:
- 安全配置基线
- 及时更新补丁
- 禁用调试模式

## A06:2021 - 易受攻击和过时的组件

使用的组件存在已知漏洞。

**防御措施**:
- 定期检查依赖更新
- 使用漏洞扫描工具
- 移除未使用的依赖

## A07:2021 - 识别和身份验证失败

身份验证和会话管理缺陷。

**常见问题**:
- 弱密码策略
- 会话 ID 暴露
- 登录尝试无限制

**防御措施**:
- 强密码策略
- 多因素认证
- 会话超时

## A08:2021 - 软件和数据完整性失败

软件更新、关键数据和 CI/CD 流程完整性受损。

**常见问题**:
- 不安全的反序列化
- CI/CD 管道配置错误
- 依赖混淆攻击

**防御措施**:
- 验证数据完整性
- 安全 CI/CD 配置
- 使用签名

## A09:2021 - 安全日志和监控失败

安全事件未被记录和检测。

**防御措施**:
- 记录登录失败
- 记录访问控制失败
- 设置监控告警

## A10:2021 - 服务器端请求伪造 (SSRF)

服务器执行未验证的 URL 请求。

**防御措施**:
- 验证 URL 来源
- 禁用 HTTP 重定向
- 白名单机制
